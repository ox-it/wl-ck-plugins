Folder Listing

This plugin allows you to embed a folder listing of a folder in the Weblearn Resources directory (from either Your Workspace or a site that you are a member of).

Clicking on a file will open the file and clicking a folder will expand the folder tree to show its contents. Users cannot go higher up the folder hierarchy than the embedded folder.

  1. Click on the Folder Listing icon.

  2. A dialog will appear with a dropdown allowing you to select from the sites available to you.

    Select the desired site and its Resources folder will be shown.

  3. Navigate through the folders by clicking on them to expand their contents until you find the folder that you wish to embed.

  4. You are given 3 pre-ticked display options for how your embed will look and function:

    • Copyright: displays a © symbol beside any file that has its copyright message set in Weblearn and displays the copyright type.

    • Descriptions: displays an information icon beside any file that has a description in Weblearn and displays the description when the icon is hovered.

    • # files in folder: displays the number of files in a folder before clicking on the folder

  5. Click OK to embed the listing.

  6. If you wish to change any of the options or the directory, right-click the placeholder and click 'Change Directory'.
