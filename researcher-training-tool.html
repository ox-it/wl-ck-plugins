Researcher Training Tool

This plugin provides a user interface for configuring and embedding the Oxford Courses Widget (sometimes referred to as the Researcher Training Tool, as it is used to display courses for graduates and researchers).

  1. Click on the Researcher Training Tool icon.

  2. A dialog will appear, giving you the full configuration options for the training tool.

    Click the Preview tab to see how the widget will be output on the page.

  3. When you are finished configuring the courses, click OK to embed the tool.

  4. If you wish to configure the tool after emedding it, right-click the placeholder and click "Configure Courses".
